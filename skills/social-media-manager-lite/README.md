## Quick Start

### Installation

Drop this folder into your Claude Code skills directory:

```bash
# macOS / Linux
cp -r social-media-manager-lite ~/.claude/skills/

# Or symlink
ln -s $(pwd)/social-media-manager-lite ~/.claude/skills/
```

```
"Crea un piano Instagram settimanale per il mio bar a Roma.
Target: professionisti 25-45, tono moderno e informale."
```

Claude will generate:
- Business snapshot

- 5-6 complete posts with captions, hashtags, timing, visuals
- Weekly summary with strategy insights
