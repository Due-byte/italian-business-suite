## GDPR Basics (Summary)

This LITE skill checks for the most common compliance issues:

1. **Privacy Policy** - You need one (visible and updated)
2. **Consent** - You need permission before collecting/using personal data
3. **Technical Cookies** - No consent needed (session, shopping cart)
4. **Tracking Cookies** - Consent required (Google Analytics, Facebook Pixel)
5. **Data Retention** - Know how long you keep data (delete after)
6. **User Rights** - People can ask for their data or deletion
7. **Security** - Protect personal data appropriately

**Key Penalties:**
- Up to ГўВ‚В¬20 million or 4% annual revenue for serious violations
- Italian Garante Privacy enforces it
